# Airline Management System Database

This project is a MySQL-based Airline Management System designed to manage and organize airline-related information efficiently. The database handles core airline operations such as passenger details, flight information, pilot records, ticket bookings, and airline scheduling.

## Features
- Passenger information management
- Flight details and route tracking
- Pilot record management
- Ticket booking and reservation handling
- Relational database structure using Primary and Foreign Keys
- SQL queries for data retrieval and analysis

## Technologies Used
- MySQL
- SQL (DDL, DML, DQL)
- Database Design Concepts

## Project Objectives
- Build a structured airline database system
- Implement relationships between multiple tables
- Perform data manipulation and retrieval using SQL queries
- Practice real-world database management concepts

This project demonstrates database normalization, relationships, joins, aggregate functions, and efficient data management techniques in SQL.


# Bookstore Management System Database

This project is a MySQL-based Bookstore Management System developed to manage book-related operations and maintain organized records of authors, books, customers, orders, and sales information.

## Features
- Author and book information management
- Customer details storage
- Order and purchase tracking
- Inventory and sales record maintenance
- Relational database design with table relationships
- SQL queries for analysis and reporting

## Technologies Used
- MySQL
- SQL (DDL, DML, DQL)
- Relational Database Concepts

## Project Objectives
- Design a real-world bookstore database
- Create relationships using Primary and Foreign Keys
- Implement SQL operations for efficient data handling
- Practice database querying and reporting techniques

This project showcases practical SQL concepts including joins, aggregate functions, filtering operations, ordering, and database management practices.
